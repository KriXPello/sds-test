# Описание
Тестовое задание для компании "Смарт Дельта Системс"


# Подготовка
- Установить зависимости командой `npm i` или `npm ci`
- Создать файл `.env` и скопировать в него содержимое `.env.example`, заполнить необходимые переменные


# Запуск в режиме разработки
- При необходимости создать файл `.env.development`
  - работает только при `mode=development`
  - при совпадении имен переменные из `.env` будут переопределены указанными
- выполнить `npm run dev`


# Сборка
- При необходимости создать файл `.env.production`
  - работает только при `mode=production`
  - при совпадении имен переменные из `.env` будут переопределены указанными
- выполнить `npm run build` (результат будет в dist/)



# Примечания к решению

- openweathermap Geocoding API ограничен в плане локализации городов и областей, поэтому при поиске на русском в результатах можно увидеть названия на английском. Также возможно дублирование городов.
